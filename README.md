## Installation and Setup
1. Fork and clone the repo and navigate to the root directory.

2. To install dependencies:
```
npm install
```

3. To run dev-build:
```
npm run dev
```

4. To run production-build:
```
npm run build
```
5. Open http://localhost:3000 in your browser
